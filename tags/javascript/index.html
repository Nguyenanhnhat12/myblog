Lập trình Node.js: Hướng dẫn cơ bản

1. Giới thiệu về Node.js

Node.js là môi trường chạy JavaScript bên ngoài trình duyệt, sử dụng engine V8 của Chrome. Đặc điểm:

  • 🚀 Không đồng bộ: Xử lý I/O hiệu quả với event-driven
  • 🌐 Backend mạnh mẽ: Xây dựng API, server nhanh chóng
  • 📦 Hệ sinh thái npm: Hàng triệu package có sẵn
  • 🛠 Đa nền tảng: Chạy trên Windows, macOS, Linux

2. Thiết lập môi trường

  • Tải Node.js từ nodejs.org
  • Kiểm tra cài đặt: 
    node -v

Javascript Dom

Thao tác DOM trong JavaScript: Hướng dẫn cơ bản

1. Giới thiệu về DOM

DOM (Document Object Model) là mô hình đối tượng tài liệu, cho phép JavaScript tương tác với HTML/CSS:

  • 🌐 Cây DOM: Biểu diễn cấu trúc HTML
  • 🛠 Thao tác động: Thêm, sửa, xóa phần tử
  • 📡 Sự kiện: Xử lý click, input, v.v.

2. Thiết lập môi trường

  • Trình duyệt: Dùng DevTools (F12)
  • HTML file: Tạo file HTML cơ bản
  • IDE: VS Code

3. Truy cập phần tử

// Lấy phần tử theo ID
const title = document.getElementById("title");

// Lấy nhiều phần tử
const items = document.querySelectorAll(".item");

4. Thay đổi nội dung

const title = document.getElementById("title");
title.textContent = "Xin chào DOM! 👋";

// Thay đổi HTML
title.innerHTML = "<strong>Xin chào</strong> DOM!";

5. Thêm/Xóa phần tử

// Thêm phần tử
const div = document.createElement("div");
div.textContent = "Phần tử mới";
document.body.appendChild(div);

// Xóa phần tử
div.remove();

6. Xử lý sự kiện

const button = document.querySelector("button");
button.addEventListener("click", () => {
    alert("Đã click!");
});

7. Ví dụ: Tạo danh sách động

<ul id="list"></ul>
<button onclick="addItem()">Thêm</button>

function addItem() {
    const li = document.createElement("li");
    li.textContent = "Mục mới";
    document.getElementById("list").appendChild(li);
}

💡 Lưu ý: Thực hành với các sự kiện, CSS manipulation và thư viện như jQuery.

Javascript Es6

JavaScript ES6: Hướng dẫn cơ bản

1. Giới thiệu về ES6

ES6 (ECMAScript 2015) là phiên bản cải tiến lớn của JavaScript, mang đến cú pháp hiện đại và tính năng mạnh mẽ:

  • 🛠 Cú pháp ngắn gọn: let, const, arrow function
  • 📦 Module: Nhập/xuất module
  • 🔄 Destructuring: Trích xuất dữ liệu dễ dàng
  • 🌟 Spread/Rest: Xử lý mảng và đối tượng linh hoạt

2. Thiết lập môi trường

  • Trình duyệt: Hỗ trợ ES6 (Chrome, Firefox)
  • Node.js: Tải từ nodejs.org
  • IDE: VS Code, thêm Babel nếu cần chuyển đổi ES6 sang ES5

3. Let và Const

let name = "John"; // Có thể thay đổi giá trị
const age = 30; // Không thể thay đổi
