Lập trình Java Spring Boot: Hướng dẫn cơ bản cho người mới bắt đầu

1. Giới thiệu về Spring Boot

Spring Boot là một framework mạnh mẽ giúp phát triển ứng dụng Java nhanh chóng, giảm thiểu cấu hình thủ công và tích hợp sẵn các tính năng cần thiết cho ứng dụng web. Framework này được xây dựng dựa trên Spring Framework nhưng cung cấp trải nghiệm phát triển đơn giản hơn với các đặc điểm nổi bật sau:

  • ⚙️ Tự động cấu hình (Auto Configuration): Tự động thiết lập các thành phần dựa trên các phụ thuộc được thêm vào.
  • 🚀 Tích hợp sẵn Web Server: Hỗ trợ Tomcat, Jetty hoặc Undertow, không cần triển khai thủ công trên máy chủ.
  • 📦 Quản lý phụ thuộc dễ dàng: Sử dụng các “Starter” để thêm các thư viện cần thiết một cách nhanh chóng.
  • 🧱 Phù hợp cho REST API và Microservices: Lý tưởng để xây dựng các ứng dụng web hiện đại và các dịch vụ phân tán.

Spring Boot được sử dụng rộng rãi trong phát triển ứng dụng doanh nghiệp, API RESTful và kiến trúc microservices.

2. Cài đặt và tạo project

Để bắt đầu với Spring Boot, bạn cần cài đặt môi trường Java (JDK) và một IDE như IntelliJ IDEA, Eclipse hoặc Visual Studio Code. Dưới đây là hai cách phổ biến để tạo một dự án Spring Boot:

Cách 1: Sử dụng Spring Initializr

  1. Truy cập Spring Initializr.
  2. Cấu hình project:
    • Project: Maven hoặc Gradle
    • Language: Java
    • Spring Boot Version: Chọn phiên bản mới nhất (ví dụ: 3.x.x)
    • Dependencies: Thêm các phụ thuộc như Spring Web, Spring Data JPA, H2 Database
  3. Nhấn nút Generate, tải file .zip, giải nén và mở bằng IDE.

Cách 2: Sử dụng dòng lệnh với Spring CLI

Cài đặt Spring Boot CLI từ trang chính thức và chạy lệnh sau để tạo project:

spring init --dependencies=web,data-jpa,h2 --name=demo-spring-boot
cd spring-boot

Mở project trong IDE để bắt đầu phát triển.

3. Cấu trúc project cơ bản

Một project Spring Boot tiêu chuẩn có cấu trúc thư mục như sau:

spring-boot/
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/example/demo/
│   │   │       ├── DemoApplication.java
│   │   │       └── controller/
│   │   │       └── model/
│   │   │       └── service/
│   │   │       └── repository/
│   │   ├── resources/
│   │   │   ├── application.properties
│   │   │   ├── static/
│   │   │   └── templates/
├── pom.xml (hoặc build.gradle)
└── README.md
  • DemoApplication.java: Lớp chính chứa phương thức main để khởi động ứng dụng.
  • application.properties: File cấu hình cho ứng dụng (ví dụ: cấu hình database, port).
  • pom.xml: File quản lý phụ thuộc nếu sử dụng Maven.

4. Ví dụ REST API cơ bản

Dưới đây là một ví dụ về cách tạo một REST API đơn giản để quản lý danh sách “Todo” sử dụng Spring Boot:

Tạo Model

package com.example.demo.model;

public class Todo {
    private Long id;
    private String task;
    private boolean completed;

    // Constructor
    public Todo(Long id, String task, boolean completed) {
        this.id = id;
        this.task = task;
        this.completed = completed;
    }

    // Getters and Setters
    public Long getId() { return id; }
    public void setId(Long id) { this.id = id; }
    public String getTask() { return task; }
    public void setTask(String task) { this.task = task; }
    public boolean isCompleted() { return completed; }
    public void setCompleted(boolean completed) { this.completed = completed; }
}

Tạo Controller

package com.example.demo.controller;

import com.example.demo.model.Todo;
import org.springframework.web.bind.annotation.*;

import java.util.ArrayList;
import java.util.List;

@RestController
@RequestMapping("/api/todos")
public class TodoController {
    private List<Todo> todos = new ArrayList<>();

    // GET: Lấy danh sách todo
    @GetMapping
    public List<Todo> getAllTodos() {
        return todos;
    }

    // POST: Thêm một todo mới
    @PostMapping
    public Todo addTodo(@RequestBody Todo todo) {
        todos.add(todo);
        return todo;
    }
}

Cấu hình application.properties

server.port=8080
spring.application.name=demo

Chạy ứng dụng

  • Mở file DemoApplication.java và chạy phương thức main.
  • Truy cập http://localhost:8080/api/todos để kiểm tra API bằng Postman hoặc trình duyệt.

5. Kết nối với cơ sở dữ liệu (H2 Database)

Spring Boot hỗ trợ tích hợp dễ dàng với cơ sở dữ liệu. Dưới đây là cách sử dụng H2 Database (cơ sở dữ liệu in-memory):

Thêm phụ thuộc trong pom.xml

<dependency>
    <groupId>com.h2database</groupId>
    <artifactId>h2</artifactId>
    <scope>runtime</scope>
</dependency>

Cấu hình application.properties

spring.datasource.url=jdbc:h2:mem:testdb
spring.datasource.driverClassName=org.h2.Driver
spring.datasource.username=sa
spring.datasource.password=
spring.jpa.database-platform=org.hibernate.dialect.H2Dialect
spring.h2.console.enabled=true

Tạo Entity

package com.example.demo.model;

import jakarta.persistence.Entity;
import jakarta.persistence.Id;

@Entity
public class Todo {
    @Id
    private Long id;
    private String task;
    private boolean completed;

    // Constructor, Getters and Setters (giống ví dụ trên)
}

Tạo Repository

package com.example.demo.repository;

import com.example.demo.model.Todo;
import org.springframework.data.jpa.repository.JpaRepository;

public interface TodoRepository extends JpaRepository<Todo, Long> {
}

Cập nhật Controller

package com.example.demo.controller;

import com.example.demo.model.Todo;
import com.example.demo.repository.TodoRepository;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/api/todos")
public class TodoController {
    @Autowired
    private TodoRepository todoRepository;

    @GetMapping
    public List<Todo> getAllTodos() {
        return todoRepository.findAll();
    }

    @PostMapping
    public Todo addTodo(@RequestBody Todo todo) {
        return todoRepository.save(todo);
    }
}

Truy cập http://localhost:8080/h2-console để kiểm tra dữ liệu trong H2 Database.

6. Kiểm thử ứng dụng

Spring Boot cung cấp công cụ để kiểm thử ứng dụng dễ dàng. Ví dụ về một bài kiểm thử đơn giản:

package com.example.demo.controller;

import org.junit.jupiter.api.Test;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.boot.test.context.SpringBootTest;

import static org.junit.jupiter.api.Assertions.assertNotNull;

@SpringBootTest
public class TodoControllerTest {
    @Autowired
    private TodoController todoController;

    @Test
    public void contextLoads() {
        assertNotNull(todoController);
    }
}

7. Triển khai ứng dụng

Để triển khai ứng dụng Spring Boot:

  1. Build file JAR:

    mvn clean package

  2. Chạy file JAR:

    java -jar target/demo-0.0.1-SNAPSHOT.jar

  3. Triển khai lên server: Sử dụng các dịch vụ như Heroku, AWS, hoặc Docker.

💡 Lưu ý: Bài viết này chỉ giới thiệu các khái niệm cơ bản của Spring Boot. Để trở thành lập trình viên Spring Boot chuyên nghiệp, bạn cần:

  • Tìm hiểu sâu về Spring Data, Spring Security
  • Nắm vững kiến trúc Microservices
  • Thực hành xây dựng các API phức tạp hơn
  • Học về triển khai ứng dụng trên cloud
